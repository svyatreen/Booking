## 🌐 Frontend Architecture (FSD - Feature-Sliced Design)

Use Feature-Sliced Design structure:

/client/src
/app (app initialization, providers, routing)
/pages (route-level pages)
/widgets (large UI blocks, e.g. hotel list)
/features (user actions: auth, booking, favorites)
/entities (business entities: user, hotel, booking)
/shared (UI kit, utils, API, config)

Requirements:

* Use separation by business logic (not by type)
* Reusable components in shared/ui
* API logic in shared/api
* State management via React Context (for auth) or hooks
